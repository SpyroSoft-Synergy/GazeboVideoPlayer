--- How to run example ---
Step by step:
1. Build plugin and source setup.bash
2. Copy "video_converter.py" to "example" folder
3. Run "python3 video_converter.py"
4. Copy all content of "script.txt" into "display_model/display.sdf" to line 17
5. Being in "example" folder run: "gazebo test_world.world"
